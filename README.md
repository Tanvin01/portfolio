# Tanvin Kheni — Portfolio

Personal portfolio site built with Next.js 14, Framer Motion, and Tailwind CSS.

## Customise in 20 minutes

**Everything lives in one file:**

```
data/portfolio.ts
```

Open it and update:
- `PERSON` — your name, bio, photo path, social links, resume path
- `EXPERIENCE` — your work history
- `PROJECTS` — your projects (add `live` URL once deployed)
- `SKILLS` — your tech stack
- `STATS` — headline numbers
- `SEO` — meta title, description, your domain

## Add your assets

Place these files in `/public/`:
- `photo.jpg` — your profile photo (square, at least 400×400)
- `Tanvin_Kheni_Resume.pdf` — your resume PDF
- `projects/devhire.png` — project screenshot (optional, shows fallback if missing)
- `projects/shopforge.png`
- `projects/chatnest.png`
- `projects/ai-resume.png`
- `projects/authvault.png`
- `projects/spendsense.png`
- `projects/logitrack.png`
- `projects/medibook.png`
- `projects/codecollab.png`

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel (free)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import repo
3. Click Deploy — done
4. Your site is live at `tanvin-kheni-portfolio.vercel.app`
5. Optional: add custom domain `tanvinkheni.dev` (~$12/yr on Namecheap)

## SEO — ranking for your name

Once deployed, do these 3 things:
1. Submit your URL to [Google Search Console](https://search.google.com/search-console)
2. Add the URL to your LinkedIn profile
3. Add the URL to your GitHub profile README

Google will index your name within 1–2 weeks.
